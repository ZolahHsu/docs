---
title:"安裝與啟動常見問題"
---

<AccordionGroup>
  <Accordion title="Q1：輸入序號後，跳出【網路連線問題，無法登入，是否再試一次】？">
    會發生此原因可能有以下兩種：

    <Steps>
    <Step title="尚未登入 Adobe 帳號">
      請先登入 Adobe，然後重新開啟 Illustrator 再試一次。
    </Step>

    <Step title="電腦或網路端的防火牆阻擋連線">
      若您有安裝防毒軟體或防火牆，請先暫時關閉後測試，也可以嘗試更換網路環境，確認是否為網路問題。
    </Step>
    </Steps>

    <Tip>
      若確認為防火牆阻擋導致，請先稍候下次更新，我們將調整網路連線方式，以降低被防火牆阻擋的情況。造成不便，敬請見諒。
    </Tip>
  </Accordion>
</AccordionGroup>

<AccordionGroup>
  <Accordion title="Q2：已安裝完 IllustKit，但開啟後沒有跳出輸入序號的對話框？">
    已發現之前推出的[【 Plugin 外掛 】](https://totallypic.com/digital-downloads/categories/plugins)可能與 IllustKit 發生衝突，導致無法正常運作。建議先暫時移除先前的外掛。
  </Accordion>
</AccordionGroup>

<AccordionGroup>
  <Accordion title="Q3：無法將 aip 檔複製到 Plug-ins 資料夾，出現權限不足的提示？">
    ▪ 可能原因：系統未授予管理員權限 ( 常見於 macOS )。<br />
    ▪ 解決方式：<br />
    1.請嘗試以系統管理員權限操作。<br />
    2.改用「macOS 安裝方法一」，透過 Illustrator 偏好設定指定外掛所在資料夾。
  </Accordion>
</AccordionGroup>
