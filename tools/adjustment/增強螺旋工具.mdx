**1\. 工具定位**

增強旋轉工具可自訂旋轉軸心位置，並在旋轉物件時同步保留筆刷角度、效果與特殊漸層的方向性，讓旋轉行為與視覺表現保持一致。

**2\. 使用時機**

- 非以物件中心旋轉，需要自訂旋轉支點時。
- 旋轉筆刷路徑時，希望筆刷方向同步旋轉。

**3\. 基本操作方式**

選擇物件後接著使用增強旋轉工具，自訂好旋轉軸心後即可旋轉物件。

- 自訂軸心位置：可拖曳軸心自訂位置，軸心位置會儲存，下次旋轉仍可使用。
- 將軸心恢復至物件邊框中心：按住 Shift 點一下軸心。
- 旋轉筆刷物件：當旋轉筆刷物件時，筆刷角度會跟隨路徑旋轉。
- 支援 效果 \> 風格化 中的「 製作陰影 」和「 塗抹 」兩種效果，效果會跟隨物件旋轉。
- 可旋轉 IllustKit 中的「 角度漸層 」、「 多邊形漸層 」。

#### **※ 原生旋轉工具 vs 增強旋轉工具**

| 項目               | Illustrator 原生旋轉工具 | IllustKit 增強旋轉工具                    |
| :----------------- | :----------------------- | :---------------------------------------- |
| **旋轉軸心設定**   | 每次需重新設定           | 可自訂軸心位置並儲存，下次沿用            |
| **回到中心軸心**   | 手動調整                 | 按住 Shift 點擊即可回到物件邊框中心       |
| **筆刷類型支援**   | 旋轉後筆刷角度可能錯位   | 筆刷角度會隨路徑同步旋轉                  |
| **效果同步旋轉**   | 需重新調整               | 「 製作陰影 」和「 塗抹 」效果 會跟隨旋轉 |
| **特殊漸層支援**   | x                        | 可旋轉 IllustKit 角度漸層、多邊形漸層     |
| **適合的使用情境** | 基本物件旋轉             | 筆刷、效果與進階漸層旋轉                  |
